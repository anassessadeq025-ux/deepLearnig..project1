# Deep Learning – Projet PyTorch

Projet de deep learning en trois parties, implémenté avec PyTorch et Jupyter Notebook.

## Contenu

| Notebook | Sujet | Dataset |
|----------|-------|---------|
| `01_partie1_mlp...ipynb` | MLP, initialisations, sauvegarde/chargement | Breast Cancer Wisconsin |
| `02_partie2_cnn....ipynb` | CNN LeNet, pooling, feature maps | Fashion-MNIST |
| `03_partie3_rnn....ipynb` | RNN / LSTM / GRU, Seq2Seq, BLEU | Tatoeba (fra → eng) |

## Installation

```bash
pip install -r requirements.txt
```

## Exécution

Ouvrir les notebooks dans Jupyter ou VS Code et exécuter les cellules dans l'ordre. Les figures sont affichées et enregistrées dans `figures/`.

## Prérequis

- Python 3.9+
- PyTorch (CPU ou GPU)
